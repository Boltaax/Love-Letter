How to run the project:

Requirements: python 3.10 or higher

Steps:

1. Inside the project folder, create a virtual environment using `python -m venv venv`
2. Activate the virtual environment using `venv\Scripts\activate`
3. Install the requirements using `pip install -r requirements.txt`
4. run the main.py file using `python main.py`

Informations about the project:

- The verbose mode is available only when playing 1 game
- Avoid to choose a high number of minmax depth, it will take a lot of time to compute the best move (1 or 2 is recommended)
